INTRODUCTION
I have created this website keeping in mind the Wikipedia page : List of Harry Potter characters. The Wikipedia page does not have any visual, but is full of valuable information. So, I created this website for all the die-hearted Harry Potter fans so that whenever they need to search for any Harry Potter character, they can look upto this website and can dig in. Whenever reading about a character, you can click on More about the Character which will direct you to the "Fandom" page of the Harry Potter Character and you will be able to know more about the character.
I have only tried to replicate the Wikipedia page mentioned above in a visually appealing way. I have made this website not to disrespect any of the websites mentioned above. I have only made this as a part of my own basic project (it is not a full-fledged working website, this is a static website) for fun and enjoyment, as I was getting bored and thought of creating something useful while learning Web Development.

NOTE : When I was making this website, I was still a beginner and have only learned HTML, CSS and basics of JavaScript. So, pardon me for any mistakes that may remain.
